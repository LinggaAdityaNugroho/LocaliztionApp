# React Project

## Description

This project is a **React JS** application built for modern web development using JavaScript.

---

## Prerequisites

Before getting started, make sure you have the following installed on your machine:

- **Node.js** (LTS version recommended)
- **npm** or **yarn**
- **Git**

Check your installation by running:

````bash
node -v
npm -v
git --version

---

## Installation Steps

### Clone Repository
Clone this repository to your local machine:
```bash
git clone https://github.com/LinggaAdityaNugroho/LocaliztionApp.git
````

Navigate to the project directory:

```bash
cd LocalizationApp
```

---

### Install Dependencies

Install all required dependencies:

Use **npm**:

```bash
npm install
```

Or use **yarn**:

```bash
yarn install
```

---

### Run Project

Start the React application in development mode:

Use **npm**:

```bash
npm run dev
```

Use **yarn**:

```bash
yarn dev
```

The application will be available at:

```
http://localhost:3000
```
